Bootstrap UI CSS Framework

Bootstrap is a powerful frontend utility ecosystem providing rapid responsive design capabilities using pre-compiled fluid components and layout classes.

The 12-Column Layout Engine

Bootstrap establishes page elements using structural system wrappers split across exactly 12 imaginary grid divisions per row, dynamically sizing across desktop screens down to standard smartphones.
