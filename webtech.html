Introduction to Web Tech

Web technologies refer to the mechanisms and tools by which computers communicate with each other using markup languages and multimedia packages.

Web Architecture

The core web architecture relies heavily on the Client-Server model. Clients request resources, and web servers process those requests.

HTML5

HyperText Markup Language is the backbone of any website. It structures the web content, defining elements and layout frameworks.

CSS3

Cascading Style Sheets control the visual presentation of a webpage, transforming plain layouts into unique user interfaces.

Bootstrap

Bootstrap is a feature-packed toolkit. It allows developers to build mobile-first, highly responsive interfaces quickly.

React JS

Developed by Meta, React is a component-based JavaScript library used for building highly interactive, dynamic frontend UIs.

Node.js

An open-source JavaScript runtime environment built on V8. It enables developers to execute high-speed server-side scripts.

MongoDB

A leading NoSQL database designed for modern apps. It stores data records in highly flexible, schemas-free JSON-like documents.
